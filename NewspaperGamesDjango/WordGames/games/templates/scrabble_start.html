Let the games begin !

Game Setup

  • Each player receives 8 random letters
  • This is a multi-player & multi-round game
  • Players take turns creating words from their letters
  • All words must be valid English words
  • Game continues until players choose to exit

Scoring System

  • Each letter has a specific point value
  • Your score is the sum of all letter values in your word
  • Player with the maximum total points wins
  • Invalid words score 0 points
  • Scores accumulate across all rounds
  • Final scores are displayed when game ends

Word Validation Rules

  • Words must be valid English dictionary words
  • Only alphabetic characters are allowed
  • You can only use the 8 letters given to you
  • Each letter can only be used once per word
  • Invalid words result in 0 points for that turn
  • Word validation uses English (US) dictionary

üéØ Game Overview:

This Scrabble simulation gives each player 8 random letters per round. Using these letters, create the highest-scoring valid English words. Points are calculated by adding up the individual letter values. The game continues for multiple rounds until players choose to exit, and the player with the highest cumulative score wins!
